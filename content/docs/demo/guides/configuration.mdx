---
title: Configuration
description: Configure the library via code or environment variables
---

## Via Code

```python
import yourlib as yl

yl.configure(
    cache_dir="~/.cache/yourlib",
    num_workers=4,
    log_level="info",
)
```

## Via Environment Variables

```bash
export YOURLIB_CACHE_DIR=~/.cache/yourlib
export YOURLIB_NUM_WORKERS=4
export YOURLIB_LOG_LEVEL=info
```

## Config File (optional)

Create `yourlib.toml`:

```toml
[core]
cache_dir = "~/.cache/yourlib"
num_workers = 4
log_level = "info"
```
